## Быстрая установка

### Windows

Откройте командную строку и выполните:

```cmd
curl -fsSL -o %TEMP%\install-mewen.bat https://raw.githubusercontent.com/ВАШ_USERNAME/mewen/main/install-mewen.bat && %TEMP%\install-mewen.bat
```

Или скачайте [install-mewen.bat](install-mewen.bat) и запустите его.

## Запуск

После установки просто введите:

```cmd
Mewen
```

## Для разработчиков

### Ручная установка

```bash
# Клонировать репозиторий
git clone https://github.com/ВАШ_USERNAME/mewen.git
cd mewen

# Установить зависимости
pip install -r requirements.txt

# Запустить
python mewen.py
```

### Структура проекта

```
mewen/
├── mewen.py              # Основной скрипт
├── requirements.txt      # Зависимости
├── install-mewen.bat     # Установщик для Windows
└── README.md            # Документация
```
